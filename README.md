PersonalExpenseSystem

Sistema di gestione delle spese personali e del budget mensile sviluppato come progetto per il corso di Programmazione / Basi di Dati.

L’applicazione consente all’utente di registrare le proprie spese, suddividerle per categoria e confrontarle con un budget mensile prefissato, attraverso un semplice menu testuale.

Funzionalità principali
	•	inserimento di nuove spese con data, importo, categoria e descrizione;
	•	definizione di un budget mensile per ciascuna categoria;
	•	visualizzazione di report riepilogativi delle spese;
	•	confronto tra spese mensili e budget impostato.

Struttura del repository
	•	src/ : codice sorgente dell’applicazione Python;
	•	sql/ : script SQL per la creazione del database e delle tabelle;
	•	demo/ : video dimostrativo del funzionamento dell’applicazione;
	•	README.md : descrizione del progetto.

Esecuzione
Per avviare l’applicazione è sufficiente eseguire il file main.py presente nella cartella src.
